# TaskPM Web UI

这是 TaskPM 的可选 Web 管理界面，基于 React 和 shadcn/ui 构建。

## 功能特性

- Worker 管理和监控
- 任务列表和状态查看
- 队列统计和管理
- 实时数据可视化

## 开发

```bash
# 安装依赖
pnpm install

# 启动开发服务器
pnpm dev

# 构建生产版本
pnpm build
```

## 说明

此 Web UI 是可选的，TaskPM 核心功能通过 API 提供。您可以：

1. 使用此 Web UI 进行可视化管理
2. 直接调用 HTTP API
3. 开发自己的管理界面

## API 文档

访问 http://localhost:28080/swagger/index.html 查看完整的 API 文档。
